    介绍

    在YuanCloud中，批次号和序列号在系统中有者相同的逻辑但是不同的表现行为。他们都是在**库存**，采购，[UNKNOWN NODE problematic]销售**模块中使用。

    [UNKNOWN NODE problematic]批次**和一定数量的收到并放在一起的产品相随。

    [UNKNOWN NODE problematic]序列号**是一个单独产品的唯一标识码，能让我们从收货到发货再到售后进行追踪。

    用到的时候

    [UNKNOWN NODE problematic]批次号**通常在大批量的产品收货中使用，这能够帮你在报表，质检，或者其余信息中反映出来。批次号能定义一些产品有或者在同一生产环境中。在食品或者服装行业的批量生产中很有用。

    [UNKNOWN NODE problematic]序列号**对于那些需要售后的产品来说比较有用，例如手机，笔记本电脑，以及其余的电子设备。依赖于管理产品的方式，你可以使用生成序列号或者自己的序列号。

    不用的时候

    消耗品例如厨房用纸，厕所用纸，笔或者手写纸等使用批次是没有意义的，因为这些产品你从来不会退货。

    另一方面，只有那些使用质保期现／或者售后服务的产品使用单个产品的序列号，除了这些，给产品分配序列号是浪费时间的行为。例如给一块面包分配一个序列号是没有一点意义的。
