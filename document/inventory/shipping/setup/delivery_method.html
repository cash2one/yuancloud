    概述

    YuanCloud可以处理不同的交货方式，但是该功能在默认设置下没有激活。交货方式可以用在销售订单上，交货单上还可以用在电商中。

    交货方式能让你管理货代供应商的公司，价格以及目的地。你甚至还可以集成YuanCloud和外部的货运公司来计算实际价格和包装

    配置

    安装存货模块

    交货方式是由**发货成本**模块处理的。进入**APP**并搜索该模块。s搜索的时候你需要在过滤栏位移除**APP**：

    注解

    If you want to integrate delivery methods in your e-commerce, you'll have to install the eCommerce Delivery module.

    配置发货方式

    要配置交货方式，进入**库存**模块，点击:[UNKNOWN NODE problematic]配置 --> 交货方式 [UNKNOWN NODE problematic]。

    首先设置一个名称以及一个物流公司

    然后你必须要设置价格。它可以被修复或者基于规则。

    • 如果价格是固定的，点选**固定价格**。你只需要定义一个价格。如果你想要在在达到一定数量时免运费，点选**如果总量大于某一点免运费**并设置一个价格。

    • 如果价格根据规则不同而不同，点选**基于规则**。点击**添加一个价格规则的新条目**。选择一个或者基于重量，容积，价格或者数量的条件。

    最终我们可以把发货方式局限在几个目的位置。你可以把它们限制为一些国家，省份甚至邮编。这种特性限制了电商的国家列表。

    注解

    你可以把YuanCloud和外部的货代公司进行集成，这样可以计算实际的价格，包装，以及处理打印装箱单的标签等。参照：[UNKNOWN NODE title_reference]

    发货流程

    销售订单

    现在你吗可以在销售订单上选择**交货方式**。如果你想在销售订单上对运费进行开票，，点击**设置价格**，它就会把交货方式作为一种产品名称添加到订单行上。

    发货

    你可以在发货单上添加或者变更发货方法

    在发货单上，查看**货代信息**。货代就是发货方式
