    为了能在YuanCloud中使用UPS APIs，你需要先有下列清单：

    • UPS.com的用户ID和密码

    • UPS账号

    • 访问密钥

    访问密钥是一个16字符的字母数字代码，允许访问的窗口开发工具包的开发和生产服务器。

    创建UPS账户

    要用MyUPS.com注册，请遵循以下步骤：

    1．用`www.ups.com <http://www.ups.com/>`登入UPS.com网站并在页面上方点击**新用户**。

    2．点击**MyUPS注册**按钮，按照以下流程完成注册过程。

    要打开UPS的账户并接收UPS运单号，请参照以下步骤：

    http://www.ups.com/content/de/en/contact/index.html?WT.svl=Footer

    一旦你创建了UPS.com 用户 ID架构，就需要把你的UPS账号添加到你的UPS.com 架构中，请参照如下步骤：

    1．使用UPS.com的用户ID和密码登陆UPS.com网站 (http://www.ups.com)

    1. 点击**My UPS**页签。

    2. 点击 [UNKNOWN NODE problematic]账户汇总**链接。

    4．在该页的**UPS 账户详情** 部分点击 添加现有的UPS账户 链接。

    5．在**添加新的账号**页面，输入**UPS 账号**，账户名，以及**邮政编码**等字段。默认的国家是美国。如果需要的话，选择**该付款账户作为默认账户**链接来设置该账户作为默认的付款方式。

    1. 点击 [UNKNOWN NODE problematic]下一个**按钮继续。

    获取访问密匙

    在完成添加UPS 账户到用户ID之后，你可以按照以下步骤从UPS请求一个访问密匙：

    1. 使用用户ID和密码登陆UPS.com。

    2. Go to the UPS Developer Kit web page https://www.ups.com/upsdeveloperkit?loc=en_US

    1. 点击 ** 请求访问密匙** 。

    2. 确认 ** 主要联系方式 **

    3. 输入 ** 第二联系方式 **

    4. 点击 ** 请求访问密匙 ** 按钮。

    一旦完成，你就能在网站页面看到**访问密匙**，带有访问密匙的电子邮件会发到你之前输入的用于联系的电子邮件中。
