    概述

    有两种方式给运费开票：

    1. 和客户达成一个协议成本价并在销售订单中固定下来

    2. 按照运输的实际成本发货。

    配置

    要给交货方式配置价格。进入**库存**应用，点击：[UNKNOWN NODE title_reference]。

    你可以手工的给运费设置价格：可以是固定的也可以根据相关规则。

    或者你可以使用货运公司的计算系统。点击文档：[UNKNOWN NODE title_reference]

    如何就运费成本给客户开票？

    按销售订单上的价格开票

    在销售订单上，选择正确的货代。点击并选择正确的**发货方式**。

    当你点击**保存**销售订单或者点击**设置价格**的时候计算价格。

    要给销售订单上的交货价格开票，点击**设置价格**，它会在销售订单上添加一个交货方式名称的产品。价格和实际价格可能会不同。

    当你创建发票的时候，发票的金额会采用订单上的金额。

    给运费的实际成本开票

    在销售订单上，选择正确的货代。点击并选择正确的**发货方式**。

    价格会在你 * [UNKNOWN NODE problematic]保存 * [UNKNOWN NODE problematic]订单的时候计算出来。确认订单并且完成发货。

    在发货单确认的时候，实际的成本就会自动计算。

    回到销售订单，实际成本已经添加到销售订单中。

    当你创建发票的时候，发票金额会用货代计算的金额。

    注解

    如果你把发货单分开并且分成好几个，每一个发货单给订单添加一行。
