    概述

    YuanCloud可以处理不同的交货方式，包含与第三方的货代的追踪信息集成。它可以让你管理货运公司，运输的实际价格和目的地。最终，你可以在YuanCloud直接打印运单上的标签。

    配置

    安装货代公司的接口模块

    在**库存**模块，点击：[UNKNOWN NODE title_reference]，在**货代接口**的下方标识你想要集成的货运公司：

    点击 应用 。

    配置发货方式

    要配置交货方式，进入**库存**模块，点击:[UNKNOWN NODE problematic]配置 --> 交货方式 [UNKNOWN NODE problematic]。

    选定货代的发货方式已经被自动创建。

    在**价格**页面，货代供应商的名称意味着该发货单会被货代的系统处理和计算。

    货代的配置被分成两栏：

    • 第一个和**你的账户**链接(开发关键字，密码，……)。更多信息请参见货运供应商的网站。

    • 第二栏根据**供应者**的不同而有所差别。你可以选择包材类型，服务类型，单位重量……

    • 一旦完成测试，就不要勾选**测试模式**。

    公司配置

    为了计算正确的价格，供应商需要知道你的公司信息。确保地址和电话号码的信息都是正确的。

    要查看你的信息，进入**设置**应用并点选**通用设置**。点击第一个链接**配置公司信息**。

    产品配置

    货代公司需要产品的重量，不然运输价格不能计算。

    进入 [UNKNOWN NODE problematic]销售**模块，点击：[UNKNOWN NODE title_reference]。打开需要发货的产品并维护重量。

    小技巧

    产品页面上的重量的默认计量单位是千克。如果使用英制计量单位的话，不要忘了单位之间的转换。

    如何打印发货标签？

    从销售订单创建的发货单会从销售订单带出发货信息，但是如果你需要的话，你可以变更货代。

    如果你在存货模块下创建一个发货转移，你可以在额外的信息页签中添加第三方货代信息。

    点击**确认**接收追踪号和**标签(可以多张)**.

    黏贴在包裹上的标签可以在下方的历史信息中找到：

    点击打开文档并打印：

    注解

    如果你在使用多包裹发货功能，大多数情况下，每一个包裹一个标签。每张标签都可以显示在交货历史中。
